Where to Learn More

API Documentation and Guides

  • Online you can lookup API documentation at apidock.com or gotapi.com
  • You can build the API doc locally with “rake doc:rails” and access them at doc/api/index.html
  • You can build the Rails guides locally with “rake doc:guides” and access them at doc/guides/index.html
